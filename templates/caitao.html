1,扫描小淘儿游戏

剩余游戏机会

get /activity/caitao_today_chances

参数名 类型 是否必须 说明
user_id int 是 -

返回：

{
    code: 0,
    data: {
        chances: 1, # 剩余游戏机会
        share: 1, # 第一次成功分享(0没有，2成功分享多次)
    }
}

游戏分享

get /activity/caitao_game_share

参数名 类型 是否必须 说明
user_id int 是 -

返回：

{
    code: 0
}

游戏开始减去机会

get /activity/caitao_game_start

参数名 类型 是否必须 说明
user_id int 是 -

返回：

{
    code: 0
}

游戏结束红包发放

get /activity/caitao_game_over

参数名 类型 是否必须 说明
user_id int 是 -
level string 是 用户分数的base64编码

返回：

{
    code: 0,
    data:[
        "满30减3",
        "满20减2"
    ]
}
